# Job Portal Analysis using Tableau Public

## Project Overview

This project analyzes a Job Portal dataset using Tableau Public.

The objective is to generate meaningful business insights by applying different filters, calculated fields, and visualizations.

---

## Tools Used

- Tableau Public 2025.3
- CSV Dataset
- GitHub
- Google Drive

---

## Tasks Completed

### Task 1
Preference vs Intern Count (Bar Chart)

### Task 2
Company Size vs Company Name (Scatter Plot)

### Task 3
Top Companies Analysis (Treemap)

### Task 4
Qualification Drilldown Map (Map)

### Task 5
India vs Germany Comparison (Stacked Bar)

### Task 6
Work Type Salary Distribution (Box Plot)

---

## Dataset

Job Descriptions Dataset (CSV)

---

## Tableau Public Dashboards

Each task has been published individually on Tableau Public.
